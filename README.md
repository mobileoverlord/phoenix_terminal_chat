# TerminalChat

Download the phoenix chat example
https://github.com/chrismccord/phoenix_chat_example

then start it
```
phoenix_chat_example> mix phoenix.server
```

then chat with it
```
phoenix_terminal_chat> mix chat
```

send messages
```
terminal: Hello
[terminal] hello
```
change your name
```
terminal: /u my name
my name: Hello
[my name] Hello
```
